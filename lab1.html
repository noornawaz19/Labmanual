You can use HTML tags to make your text look different. Here are some examples.

  • Bold text using the <b> tag. For example, don’t forget to save your work.

  • Underline text using the <u> tag. Check the due date before you submit.
  • Subscript text like H2O using the <sub> tag. Water is written as H2O in science.
  • Superscript text like x2 using the <sup> tag. In math we write x2 for x squared.

These tags are easy to use and make your webpage look nicer.
